# Encapsulation Example in Java

## Overview
This project demonstrates the Encapsulation principle in Java.

The BankAccount class protects its internal state by:
- Using private fields
- Providing controlled access methods
- Validating data before modifying state
- Preventing direct modification of balance

## Key Concepts
- Data hiding
- State protection
- Controlled access
- Business rule enforcement

## Why This Matters
Encapsulation ensures that object state cannot be modified directly.
All changes must go through validated methods (deposit and withdraw).
