# Polymorphism Example in Java

## Overview
This project demonstrates the Polymorphism principle in Java.

Different shapes (Circle and Rectangle) implement the same interface (Shape).
The program works with the abstraction type instead of concrete implementations.

## Key Concepts
- Runtime polymorphism
- Interface-based design
- Loose coupling
- Clean architecture

## Why This Matters
Polymorphism allows writing flexible and extensible systems.
New shapes can be added without modifying existing logic.
